🔐 VAULT

VAULT is an interactive cybercrime awareness and response platform designed to help users identify, understand, and respond to digital threats such as phishing, fraud, and account breaches.

🚀 Features
	•	💬 AI-style chatbot for cyber guidance
	•	🌍 Multi-language support
	•	🇨🇦🇸🇮 Country-specific assistance (Canada & Slovenia)
	•	📊 Cyber dashboard with threat stats
	•	🔥 Heatmap visualization of cybercrime intensity
	•	📰 Cybersecurity articles section
	•	🗺️ Interactive map-based reporting system
	•	🌗 Dark / Light mode UI

🧠 How It Works

Users describe their cyber issue through the chatbot.
The system analyzes the query and provides actionable advice along with relevant reporting resources based on the selected country.

🛠️ Tech Stack

Frontend
	•	React
	•	CSS (custom UI + animations)
	•	Three.js (3D globe)
	•	Leaflet (map & heatmap)

Backend
	•	FastAPI
	•	Python

  🎯 Purpose

This project was developed as a capstone to demonstrate the integration of modern web technologies, UI/UX design, and intelligent systems in the domain of cybersecurity awareness and assistance.
